# PharmaLink B2B — Supply Marketplace PoC

**Segment 2 — B2B Supply Marketplace** (Pharma Industry)

Live demo of all screens for a B2B pharma procurement mobile app.

## Screens Included

| Screen | Description |
|--------|-------------|
| Business Login + KYC | OTP login, drug license upload, business type selection |
| Home / Supplier Discovery | Live supplier listing, categories, reorder alerts |
| Live Stock & Price View | Product catalog with stock levels, bulk pricing |
| Cart & Bulk Order | Multi-item cart, bulk discount engine, delivery options |
| Payment & Credit | Credit line, UPI, NEFT, COD payment methods |
| Digital Invoice | GST-compliant invoice with batch numbers, QR verification |
| Delivery Tracking | Live map simulation, step-by-step tracking, driver info |
| Order History | All past orders with status, filter by delivery state |
| Sales Analytics Dashboard | Spend trends, top products, supplier breakdown charts |

## Deploy on Render

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New + → Static Site
3. Connect this repo
4. Publish directory: `.`
5. Click **Create Static Site**

Live URL: `https://pharmalink-b2b-poc.onrender.com`
